Presentation: (50 points) 
Create an IoT conceptual design presentation following the guidelines below. 
1. Overview 
The IoT Conceptual Design Presentation focuses on proposing a smart solution using IoT 
technologies. Students must present a well-planned system design, discussing how devices, 
sensors, and data interact to solve a real-world problem.

b. Design a conceptual IoT architecture
